Cee-Lo game

A fun dice game

I Implement a program that runs a dice game called Cee-lo. In Cee-lo, 3 dice are rolled, and a score is allocated to the dice roll depending on what values have been obtained. In Cee-lo there are only 4 valid dice rolls: • 4-5-6, 1-2-3, Trip: All 3 dice rolled have the same value. Point: 2 of the 3 dice rolled have the same value.        The program prints out a banner indicating the game has started. Then it prints out a menu letting the player choose whether to play another round of Cee-lo (the player enters 1) or to exit the game (the player enters 0). A separator is printed before a round starts, during a round a valid Cee-lo dice roll is generated for the player and for the computer. The scores allocated to the player and the computer rolls are calculated. Information on what the player and the computer have rolled is printed. The winner of the round (or if it is a draw) is printed based on the scores obtained. A separator is printed after the round ends. When the player exits the Cee-lo game, the program prints a separator. Lastly it prints how many rounds the player has won and their win percentage. 

  • Date: 2021
  • Tech Stack: Python
  • Category: Software Development
